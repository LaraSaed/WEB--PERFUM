# 🏺 Celestia — Luxury Fragrance Experience

**Celestia** is a high-end, responsive landing page designed to showcase a premium perfume brand. This project was built to demonstrate my ability to blend high-quality visual branding with clean, functional code.

### ✨ Key Features
* **Dynamic UI:** Interactive elements powered by custom JavaScript (`cart.js`, `product.js`).
* **Responsive Architecture:** A seamless experience across mobile, tablet, and desktop.
* **Modern Aesthetic:** Focused on luxury typography and a soft, elegant color palette.
* **Interactive Components:** Functional navigation and product sections.

### 🛠️ Technical Stack
* **Structure:** Semantic HTML5
* **Styling:** Custom CSS3 (Flexbox & Grid)
* **Logic:** JavaScript (ES6+)
